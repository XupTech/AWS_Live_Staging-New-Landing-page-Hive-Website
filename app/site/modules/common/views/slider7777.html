Handyforall User App Features

Let's explore the new features added to our Handyforall User Application

OTP Registration

Users can use the OTP verification to register their mobile numbers. The taskers can also sign in to the app using OTP.
User Wallet

Customers and taskers can do cashless transactions with the help of the wallet module. Customers have to load the wallet & the money gets automatically deducted after the task.
Instant Task alerts

Both the users and taskers will Get instant task alerts from the application. It helps to know the real-time status of the task.
Emoji Rating

The users can rate the tasker after work. Users can rate using emojis to get feedback.

Schedule Job

Promo Code

Admin can send Promo Codes to customers through push notifications. Using this in-app feature, admin can stay in touch with users.
